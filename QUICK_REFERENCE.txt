## Installation (Copy-Paste)

```bash
# Install Netlify CLI
npm install -g netlify-cli

# Navigate to project
cd h:\My Website\GameHUB

# Login
netlify login

# Test locally
netlify dev
```

Visit: **http://localhost:8888**

---

## Environment Variables to Create

### Root: `.env.local`
```env
VITE_API_URL=http://localhost:8888/.netlify/functions/api
MONGODB_URI=your_mongodb_url_here
JWT_SECRET=your_secret_key
JWT_EXPIRE=30d
NODE_ENV=development
```

### Frontend: `gameverse/frontend/.env.local`
```env
VITE_API_URL=http://localhost:8888/.netlify/functions/api
```

## Deploy Commands

```bash
# Test locally (MAIN COMMAND)
netlify dev

# Deploy preview
netlify deploy

# Deploy production
netlify deploy --prod

# View logs
netlify logs --tail

# Check status
netlify status
```

## Web Deployment (Easiest)

1. Go to **https://netlify.com**
2. Sign up / Login
3. "New site from Git"
4. Select your GameHUB repo
5. Build: `npm run build:netlify`
6. Publish: `gameverse/frontend/dist`
7. Set environment variables
8. Deploy! ‚úì

## Key Reminders

1. **Whitelist Database IPs** in MongoDB Atlas
2. **Set env variables** in Netlify dashboard
3. **Test locally first** with `netlify dev`
4. **Don't commit** `.env.local` files
5. **Keep backend separate** for Socket.io (if needed)

## Troubleshooting

| Problem | Solution |
|---------|----------|
| API 404 | Check `VITE_API_URL` env var |
| CORS Error | Set `FRONTEND_URL` in Netlify |
| DB Timeout | Whitelist Netlify IPs |
| Build Fails | Check `netlify logs` |
